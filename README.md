# OpenAfrika — African Open Source Foundation  > Building the open infrastructure that powers Africa's digital future.  
## What We Are 
OpenAfrika (AOSF) is a pan-African open source foundation building the  African Knowledge Commons — the world's most comprehensive open-access  repository of African language data, cultural knowledge, and lived experience.  This Commons is the foundational data layer for African AI. Open source.  Governed by Africans. Owned by no single company or government.
